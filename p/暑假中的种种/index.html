搭建Hugo 2023.07.31 22时 星期一

因为之前买的腾讯云轻量化服务器要过期了，续费的话对暑假没有生活费的我来说有些昂贵其实在四十五，因为暑假花得太多了，于是我想到了Hugo+Github Page方案，一来不需要为服务器价格发愁，而来也提高一下编程水平。

学习搭建hugo的时候，也学习了一些git的使用方法，为了加强记忆，所以我记录下来：

  • 初始化本地仓库git git init
  • 认证ssh密钥 ssh-keygen -t rsa -C "youremail@example.com"
  • 获取密钥 cat ~/.ssh/id_rsa.pub
  • 连接仓库 git remote add 仓库别名 仓库地址
  • 提交: 1. git add . 2.git commit -m '备注' 3.git push -u origin master

明天继续努力把hugo配置成想要的样子！

没有钱的痛苦 2023.08.07 10时 星期一

暑假没有钱就是最痛苦的，但其实我不是一直没有钱，而是钱花得太快了。我的娱乐活动，全靠长辈们，但也不多，老汉基本上不给我钱，很窘迫。昨天和王傲冉他们出去玩了，很快乐，但是为了去吃饭，csgo库存里值钱的全被我卖掉了，哎，不过还是现实的快乐更快乐，同时，也见到了李佳橦，一句话也没说。

第一次出来，想说又不敢说，第二次，我们高兴的牵手，现在我只有维持事宜地沉默。

昨天，在恍惚间，我想到以前做过的事：还是小学三四年纪吧，那天早上我哭闹着要我妈妈带我去上学，而不是我奶奶，并不是因为我更喜欢我妈妈而不喜欢我奶奶，而是我眼红于学校门口卖玩具，好像是十一元，对于我来说是一笔巨款，父母也不会给我零花钱，只有妈妈带我去上学，也许我哭闹才能让她给我买。但是她没空，也要去上班，应付不住我的哭闹，给了我五块钱，父亲在旁边吃早饭，很生气，我还是继续哭闹，妈妈只好带着我去学校，到了学校门口，看着玩具，妈妈拒绝了，我也只有去上学，拼命哭闹也并没有得到什么，回过头来，很遗憾，为什么我要这么执着最后大家都不高兴呢？小时候的我不懂，现在也许懂了。

开学前紧张 2023.08.21 19时 星期一

还有十天就要开学了，暑假过去得也太快了吧！ 这是大学的第二个假期，没有发生特别的事，一切都很寻常，和哥们去特色牛肉面吃早饭，晚上去宏帆 ，西湖塘逛逛，集合总在麦当劳靠窗的座位，早上还是时不时睡懒觉，下午玩电脑到打瞌睡，晚上说不熬夜还是在上床之后多刷个把小时手机。

想到上次去打乒乓已经过了很久了，但是还在暑假里，这么一想，暑假的两个月还是很漫长，不过每天都是重复。

回到学校之后，三个还不熟悉的室友，夹在中间的座位，没有能对上电波的室友，睡一晚就会揉成一团的床单，真是糟心，好窘迫。

也许，我该和新室友旧室友们增强感情。

吃麦当劳的时候我在吃什么？ 2023.08.30 23时 星期三

上一次吃麦当劳是什么时候呢？

应该已经有半年了。

昨天，我爸说有应酬，不回来吃饭，妈妈也要上班，正好我很久没吃快餐了，准确说是像麦当劳一样美味的快餐了，于是我妈说今天晚上去吃麦当劳吧。

早上八点过起来，我妈买菜回来，还带了几个包子和燕窝丝，吃的时候想起来在学校遇到的一件事（燕窝丝 or 花卷？）不过不是重点，她上班出门之前说土豆丝给我切好了，可惜早餐吃太饱，玩累了一觉睡到下午六点。

我妈的电话把我叫醒，她问我是出来吃还是她给我带回来，从睡梦中被拽醒的人面对选择是很勉强的，我说那你带回来吧。电话刚打完，睡意就结束了。

打开微信小程序，在麦当劳各种套餐里对比，果然，好贵。我从小奢望的，就是从容地在麦当劳这样的快餐店里吃饱。对比一下美团上的价格，带给我的只有窘迫，母亲发来一张截图问我怎么样，当然，美味，好少，好贵。我说我在看看划算的，正好发现一个随心配，哈哈多好笑的命名，给最不能随心的人一个随心配的选择。

还是那么好吃，袋子都是快餐店空调的好闻味道。

人们说中国人最爱反思，做什么事都要反思，但是我无力，心中羞愧。

我和朋友说：“很多年以后，我也爱吃炸鸡喝可乐。我以后最想过的生活就是一日三餐都能去快餐店吃饭。“

开学 2023.09.02 18时 星期六

昨天晚上七点出发，经过四个小时的车程终于到了学校，中途发现车票买错了，需要到成都南转站。 在车站的候车厅里，我那个检票口只有零零散散的几个人，椅子被空调吹得冷得惬意，把我妈放在书包里的冰面包拿出来，搭配一瓶阿萨姆奶茶，这是远去的我最好的晚饭。
